关于 .gitignore

2025年8月27日·672 字·4 分钟
Git .Gitignore 版本控制

项目文件纳入版本控制后进行了初始化提交，导致很多不必要文件也纳入了版本管理，这时候再添加 .gitignore 文件进行排除是无效的。

这时候我们可以这样做：

git rm -r --cached .

然后项目文件再整个重新提交

git add .

git commit -m 'commit message'

git push origin branch-name
